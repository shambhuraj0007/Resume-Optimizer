﻿# ResumeItNow

ResumeItNow is a free, open-source, AI-powered resume builder designed to help you create professional, ATS-friendly resumes with modern templates. No watermarks, no hidden fees.

---
## Live preview
https://resume-virid-iota.vercel.app/

## 🚀 Features

- **Modern, Professional, Minimal & Creative Templates**  
  Choose from a variety of stunning, ATS-optimized designs.
- **AI-Powered Suggestions**  
  Get smart content recommendations for your resume sections.
- **Live Preview & Editing**  
  Instantly see changes as you edit your resume.
- **PDF Export**  
  Download your resume as a polished PDF, ready to submit.
- **Custom Sections**  
  Add custom sections to tailor your resume.
- **No Account Needed**  
  Build and download resumes without registration.
- **User Profile & Resume Management**  
  Save, view, and delete your resumes.
- **Responsive & Accessible UI**  
  Works seamlessly on desktop and mobile devices.
- **Open Source**  
  Free to use and modify.

## 🛠️ Setup & Run Instructions

1. **Clone the repository**

   ```sh
   git clone https://github.com/shambhuraj0007/ResumeAI.git
   cd ResumeAI
   ```

2. **Install dependencies**

   ```sh
   npm install
   ```

3. **Configure environment variables**

   - Copy `.env.local.example` to `.env.local` and fill in required values (Firebase, OpenRouter API keys, etc).

4. **Run the development server**

   ```sh
   npm run dev
   ```

   - The app will be available at [http://localhost:8888](http://localhost:8888)

5. **Build for production**
   ```sh
   npm run build
   npm start
   ```
